# Web3 实习计划（冬季）- 课程大纲

由 ETHPanda 与 LXDAO 联合发起的 Web3 实习计划课程大纲网站。

## 功能特点

- 📅 课程时间表展示
- 🎥 YouTube 和 Bilibili 双平台回放链接
- 📄 课程幻灯片下载
- 🔍 课程搜索功能
- 📱 响应式设计

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无需构建工具
- 数据驱动（JSON）

## 本地开发

```bash
# 启动本地服务器
python3 -m http.server 8080

# 访问
open http://localhost:8080
```

## 更新课程数据

编辑 `data/courses.json` 文件即可。

## 部署

本项目已部署到 Vercel。
